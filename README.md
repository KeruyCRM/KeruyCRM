KeruyCRM
==
2.0 Alpha
####
Систему повністю переведено на [FatFreeFramework 3.8](https://fatfreeframework.com)

Максимально підвищена безпека системи (маппінг sql-запитів, сучасні криптоалгоритми для сесій/паролів/токенів, усі форми захищені через csrf-токени та т.і.). Гнучке кешування sql-запитів. Багатопротокольний механізм зберігання сесії та кешу. Систему повністю переведено на ООП-парадигму і MVC (де це можливо). Додано контроль цілостності внесення змін до БД через процедуру коммітів. Повністю перероблена система локалізації. 

Це безкоштовна система для управління проектами з відкритим кодом. Вона, далека від традиційних додатків, надає користувачам більш широкий і розширений підхід до управління проектами. Її параметри налаштування дозволяють користувачам створювати додаткові об’єкти, змінювати та вказувати зв’язки між ними, а також створювати необхідні звіти. Платформа надає можливість користувачам створювати власний додаток, спеціально адаптований для їх діяльності (CRM, ERP, HRM, WMS тощо).

KeruyCRM може бути реалізований не тільки як інструмент управління проектами, але і як CRM-платформа, або як комбінація того і іншого.

####
Особливості

* Конструктор баз даних
* Конструктор форм
* Конфігурація доступу
* Сповіщення користувачів
* Гнучки звіти
* Автоматизація дій
* Інтеграція с поштою
* Інтеграція з сервісами доставки SMS
* Експорт/імпорт записів
* Резервне копіювання бази даних
* Багатомовність (українська мова за замовчуванням)
* Список справ
* Інтеграція карт
* Штрих-код і QR-код
* Підтримка LDAP
* та багато іншого
